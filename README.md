# @2e32/react-select

React select.

## How install

```bash
yarn add @2e32/react-select
```

## Usage

App.js

```javascript
// Once (css import)
import '@2e32/react-select/css';

const App = () => <Page />;

export default App;
```

Page.js

```javascript
import Select from '@2e32/react-select';

const Page = () => {
  return <Select />;
};

export default Page;
```
